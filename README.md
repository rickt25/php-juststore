# Just Store | E-Commerce PHP Native
Built with :
- PHP 7.4.8
- Apache 2.4.43
- MySQL 5.0.2

External Libraries :
- Bootstrap CSS (CDN)
- Jquery Tokenize (CDN)

## Installation Guide
1. Download XAMPP for Windows.
2. Run Apache and MySQL.
3. Import SQL File from repo
4. Move project to /htdocs in xampp folder
5. Open http://localhost/php-juststore in browser
